What is PatternGen

PatternGen is a random CSS pattern generator that generates usable CSS code. The code can be saved and used on your personal web page.

This way you can easily get a unique background for your website.

How Does it Work?

PatternGen uses a random number generator to generate a 16 digit integer that is used as a seed. Each digit of the seed is implemented in creating the CSS code.

The first 6 digits are what generate the RGB values of the color of the pattern. There is a main color, which is part of the seed, and a secondary color, which is the complement of the main color (255 minus the main RGB values). The patterns themselves are created with a few generating function that use digits 7 to 16 to get the values for the CSS code. The circles in the pattern are all made using radial gradients.
